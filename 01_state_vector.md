# Biological State Vector

We propose representing the biological state of a system as

S = (x₁, x₂, ..., xₙ)

where each component represents a measurable biological observable.

Possible observables include:

- Fluorescence intensity
- UV absorbance
- FTIR features
- Raman intensity
- Electrical impedance
- Temperature

Future work:
- Define state evolution.
- Define state similarity.
