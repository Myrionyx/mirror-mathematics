## Biological State

A biological state is represented as

S ∈ Ω

where Ω denotes the biological state space.

Examples

- Native protein
- Folded state
- Cellular equilibrium
- Recovery state

---

## Perturbation

A perturbation changes one biological state into another.

P : Si → Sj

where

Si = initial state

Sj = perturbed state

---

## Mirror Response

Mirror response is defined as the biological tendency toward a new stabilized configuration.

M(Sj)=Sk

Sk does not necessarily equal Si.

It may represent a partially recovered biological state.

---

## Biological State Space

Ω={S1,S2,…,Sn}
